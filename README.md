# 🎬 React Movie App

A simple movie browsing application built using React and TMDB API.

## 📌 Description

This project allows users to:

- Browse popular movies
- Search movies
- View movie details
- Add movies to favorites
- Navigate between pages using React Router

This project was built for learning and final assignment purposes.

---

## 🛠 Tech Stack

- React
- Vite
- Axios
- React Router
- Tailwind CSS
- TMDB API
